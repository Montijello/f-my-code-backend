# __F My Code Back End__

## __Installation__

* Download from GitHub

* Run ```npm install```

* Start in "Development Mode" ```npm run dev```

## __Create Knex DataBase__

* First, ```npm install knex```

* Second, ```npm run knex migrate:latest```

* Then, ```npm run knex seed:run```
